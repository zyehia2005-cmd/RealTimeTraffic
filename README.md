# RealTimeTraffic
**RealTimeTraffic** uses data engineering pipelines to collect, clean, and integrate IoT, GPS, and sensor data in real time. It applies AI and GIS analytics to visualize and predict traffic flow, enabling smarter mobility decisions, reduced congestion, and efficient urban traffic management.
